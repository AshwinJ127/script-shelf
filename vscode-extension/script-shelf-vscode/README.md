# Script Shelf for VS Code

Connects your VS Code editor directly to your Script Shelf library.

## Features

- **ScriptShelf: Login**: Authenticate with your Script Shelf account token.
- **ScriptShelf: Save Selection**: Highlight code in your editor and save it instantly to your library.
- **ScriptShelf: Insert Snippet**: Search your remote library and insert code snippets directly at your cursor.

## Requirements

You must have a Script Shelf account. Get your auth token from your dashboard.

## Extension Settings

This extension uses 'scriptshelf.authToken' (stored securely) to authenticate requests.
